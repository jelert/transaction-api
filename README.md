# README

Joe Elert's transaction API for Fetch Rewards Backend Software Engineering

* Node.js version v18.7.0

* Dependencies:
    - Node.js
    - Express (to install with npm run `npm install express`)
    - Request (`npm install request`)

* To Run:
    - Match Node version 
    - run `node app.js`

* Services:
    - Transaction
    - Spend
    - List
